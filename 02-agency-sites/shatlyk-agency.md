# Shatlyk1011/agency-website

> **Next.js agency landing with refined motion.**

- **URL:** https://github.com/Shatlyk1011/agency-website
- **Stack:** Next.js · Framer Motion · Tailwind · GSAP

## Techniques worth stealing

- **Section-by-section pinning** — each service pinned while text changes.
- **Framer `useScroll` + `useTransform`** for parallax backgrounds.
- **Dark/light toggle with CSS variables + Framer crossfade**.
- **Custom cursor** that morphs on hover over project cards.
- **SVG-path draw-on-scroll** for decorative dividers.

## Why inspiring

If the user brief is **digital agency / consulting studio**, this is the cleanest Next.js
starting point. Fewer bells than Dennis Snellenberg but production-ready.
