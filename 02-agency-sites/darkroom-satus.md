# darkroomengineering/satus

> **A pro studio open-sourced their internal starter.** This is how agencies wire a site from zero.

- **URL:** https://github.com/darkroomengineering/satus
- **Live:** https://satus.darkroom.engineering
- **Stars:** ⭐ 600+
- **Stack:** Next.js 15 App Router · Sass Modules · Lenis · Hamo · Theatre.js-ready

## Techniques

- **Production-grade Lenis integration** via React context + single RAF orchestrator.
- **Responsive CSS variables** driven from a single `tokens.scss` file.
- **`useTempus` / `useTransform` Hamo hooks** for frame-synced animations (Hamo is Darkroom's
  own "motion glue" library).
- **Built-in view-transitions** and GSAP plug points.

## Why inspiring

Darkroom Engineering made Lenis. They also build SOTD-winning sites for clients like Linear,
Arc Browser, Bun. **Satus is the seed they start from**. If you want your project to breathe
like theirs do from day 1, clone this and start deleting.
