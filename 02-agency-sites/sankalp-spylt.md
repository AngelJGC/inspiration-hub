# Sankalp20Tiwari/spylt

> **Pixel-honest Tubik Studio "Spylt" rebuild.** Beverage-agency vibe.

- **URL:** https://github.com/Sankalp20Tiwari/spylt
- **Stars:** ⭐ 200+ (approx)
- **Stack:** React · React Router · Tailwind CSS · GSAP (ScrollSmoother, ScrollTrigger)

## Techniques worth stealing

- **ScrollSmoother inertia** — Lenis-like feel but uses GSAP's native smoother.
- **Reveal text on enter** — `y: 100 → 0` + `opacity: 0 → 1` with stagger per line.
- **Fluid section transitions** — backgrounds shift color with scroll progress.
- **Sticky product scenes** — pinned hero with foreground motion.
- **Route-based transitions** — page enter/leave animations via GSAP + React Router.

## Why inspiring

Clean, readable React. If the user wants a **drink / FMCG / snack brand** landing, this is the
fastest path from zero to credible. Every GSAP block is scoped with `gsap.context()`.
