# jagadeesh-k-2802/react-web3-landing-page

> **Clean, hackable Web3 boilerplate.** How far you can go with pure Framer + Tailwind.

- **URL:** https://github.com/jagadeesh-k-2802/react-web3-landing-page
- **Live:** https://react-web3-landing-page.vercel.app
- **Stars:** ⭐ 100+
- **Stack:** React 18 · Vite · Framer Motion · Tailwind CSS

## Techniques

- **Staggered hero entrance** using Framer `variants` with `staggerChildren`.
- **Glassmorphism cards** — `backdrop-filter: blur()` + noise overlay PNG.
- **Animated gradient mesh background** via CSS `@keyframes` on `background-position`.
- **Section-scoped `whileInView`** triggers that re-animate on re-entry.

## Why inspiring

Easy to lift section patterns into Next.js. No GSAP needed — pure Framer. Good starting point
when the brief is "Web3 landing, shipped in a week".
