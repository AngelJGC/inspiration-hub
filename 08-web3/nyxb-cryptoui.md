# nyxb/cryptoui

> **Full DeFi-style marketing page assembled with shadcn primitives.**

- **URL:** https://github.com/nyxb/cryptoui
- **Live:** https://cryptoui.nyxb.xyz
- **Stars:** ⭐ 300+
- **Stack:** Next.js 14 App Router · TypeScript · Tailwind · Framer Motion · shadcn/ui

## Techniques

- **Animated token ticker** using `motion.div` + `useMotionValue` with infinite `x` loop.
- **Scroll-linked protocol stats reveal** — `useScroll` + `useTransform`.
- **Section-pinned parallax** via CSS `position: sticky` + staggered inner translates.
- **Dark/light token-aware theming** through CSS variables.

## Why inspiring

The polish-fast combo: shadcn components + Framer motion. Great starting kit when you want
polish without rolling animation primitives from scratch.
