# btahir/react-kino

> **Drop-in cinematic components for longform editorial pieces.**

- **URL:** https://github.com/btahir/react-kino
- **Stack:** React · GSAP · IntersectionObserver

## Techniques worth stealing

Ready-made components that you import and compose:

- `<Scene>` — pinned section container
- `<Reveal>` — opacity/y fade-in on enter
- `<ScrollTransform>` — interpolate any CSS property with scroll
- `<Parallax>` — layered scroll-rate backgrounds
- `<Counter>` — number count-up on visible
- `<Marquee>` — infinite horizontal ribbon
- `<TextReveal>` — word-by-word reveal on enter

## Why inspiring

If `basement-scrollytelling` is "build your own timeline", this is "import and ship". Great
for quickly prototyping editorial articles, case studies, annual reports, documentary-style
microsites.
