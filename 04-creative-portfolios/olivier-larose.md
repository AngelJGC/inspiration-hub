# Olivier Larose — tutorial library

> **The definitive tutorial library for the Next.js + Framer Motion creative stack.**

- **URL:** https://blog.olivierlarose.com/tutorials (repos linked per-tutorial)
- **YouTube:** https://www.youtube.com/@olivierlarose1
- **Stack:** Next.js · Framer Motion · GSAP · Lenis / Locomotive · Tailwind

## Recurring tutorial patterns

| Tutorial | Technique |
| --- | --- |
| "Sticky cursor" | Magnetic follower that grows over buttons |
| "Magnetic elements" | `useMotionValue` + spring on `mousemove` |
| "Page transitions" | Full-bleed curtain wipe between routes |
| "Smooth-scroll landing" | Lenis + Framer `useScroll` parallax |
| "Awwwards landing rebuild" | Multi-section pinned narrative |
| "Horizontal scroll gallery" | Translate-X tied to vertical scroll |
| "Image carousel (Framer)" | Keyed index + AnimatePresence |
| "Text reveal on scroll" | Line-by-line stagger with intersection |
| "Infinite marquee" | `repeat: -1` animation loop |
| "Work page hover effect" | Image preview follows cursor on link hover |

## Why inspiring

Olivier's tutorials are the de-facto curriculum for anyone building a modern creative React
site. Each tutorial is a small focused repo — perfect to cherry-pick techniques.

## Usage

When the user says "make it feel like Dennis Snellenberg / Manoloide / Hello Monday", you can
almost always find a matching Olivier Larose tutorial that teaches the specific technique
needed.
