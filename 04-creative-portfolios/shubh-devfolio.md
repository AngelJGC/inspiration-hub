# shubh73/devfolio

> **Clean GSAP portfolio starter.**

- **URL:** https://github.com/shubh73/devfolio
- **Stack:** Next.js · GSAP · Tailwind · React

## Techniques worth stealing

- **ScrollTrigger timelines** — one master timeline per section, children chained.
- **SplitText hero** — animated per-word entrance.
- **Section pinning** — about / skills / projects each pin for a few viewport-heights.
- **Dark-only palette** with accent-on-hover.

## Why inspiring

If `ayush-folio` is the "polished" version, this is the "buildable in a weekend" version. Good
for a developer who wants GSAP-driven motion without the Framer Motion + Lenis overhead.
