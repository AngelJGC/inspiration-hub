# adrianhajdin/3d-portfolio

> **The canonical beginner-friendly R3F + Framer portfolio.**

- **URL:** https://github.com/adrianhajdin/3d-portfolio
- **Video:** https://www.youtube.com/watch?v=0fYi8SGA20k — "Build and Deploy an Amazing 3D Web Developer Portfolio in React JS"
- **Channel:** JavaScript Mastery (Adrian Hajdin)
- **Forks:** `sanidhyy/3d-portfolio` and many more
- **Stack:** React · Vite · Three.js · React Three Fiber · drei · Framer Motion · Tailwind · EmailJS

## Techniques worth stealing

- **Brittany Chiang–inspired layout** — the "gold standard" dev portfolio structure.
- **Animated 3D hero** — desktop/planet/island model driven by R3F.
- **Section navigation** — smooth scroll with hash-based active states.
- **Project cards** — tilt effect via `react-parallax-tilt` or custom.
- **Contact form via EmailJS** — no backend needed.
- **Framer Motion page transitions**.

## Why inspiring

One of the highest-starred repos in the "3D portfolio" category. Excellent scaffolding for a
first 3D React build. Tutorial is ~4 hours and very thorough — good for a handoff if the user
wants to iterate further themselves.
