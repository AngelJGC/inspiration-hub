# aceternity/ui

> **The quintessential "AI/SaaS landing" look.** Every block has a matching Framer implementation.

- **URL:** https://github.com/aceternity/aceternity-ui
- **Live:** https://ui.aceternity.com
- **Stars:** ⭐ 6k+
- **Stack:** Next.js App Router · Framer Motion · Tailwind · Three.js (some blocks)

## Techniques

- **Spotlight / aurora backgrounds** driven by Framer `motion.div` gradients.
- **Infinite scrollable testimonial rows** with pause-on-hover.
- **Cursor-follow background gradients** using `useMotionValue` + CSS vars.
- **Card hover 3D-perspective tilt** without external tilt libs (pure CSS transforms).
- **3D Card Effect, Bento Grid, Hero Parallax** — each is a named, copy-pasteable block.

## Why inspiring

If you've browsed modern AI startup landings (Perplexity, Cursor, Linear-style), you've seen
aceternity blocks. It's the fastest way to ship an **"AI-startup-in-2026"** aesthetic. Lives in
the same universe as shadcn and magicui.
