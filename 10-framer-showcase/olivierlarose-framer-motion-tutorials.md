> **Fastest way to internalize Framer idioms** — every folder = one popular YouTube tutorial.

- **Stars:** ⭐ 700+
- **Stack:** Next.js App Router · Framer Motion · TypeScript

## Techniques

- **Reveal text** with per-word `overflow-hidden` wrappers (words slide up from mask).
- **Dynamic page transitions** using a `<Template>` component with exit timing.
- **Magnetic buttons** via springed `useMotionValue`.
- **Preloader with split-flap character reveal** — airport departure board vibe.

## Why inspiring

Every folder matches a narrated YouTube build. You get the code + the story of why each
decision was made. This is the **Framer equivalent of the `olivier-larose` tutorial library**
in 04-creative-portfolios.
